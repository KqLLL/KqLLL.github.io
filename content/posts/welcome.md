# 欢迎来到我的博客

你好！欢迎来到我的技术博客。这是我的第一篇文章。

## 关于这个博客

这个博客使用现代化的技术栈构建：

- **框架**: Next.js 15 (App Router)
- **样式**: Tailwind CSS
- **内容**: Markdown/MDX
- **部署**: GitHub Pages

## 主要功能

这个博客支持以下功能：

1. **Markdown 文章支持**
   - 支持标准 Markdown 语法
   - 代码高亮
   - 数学公式（可选）

2. **标签和分类系统**
   - 按标签浏览文章
   - 按分类组织内容

3. **评论系统**
   - 使用 Giscus 集成 GitHub Discussions

4. **搜索功能**
   - 快速搜索文章标题和内容

5. **深色模式**
   - 支持亮色/深色主题切换
